我们是谁

我们是一支来自于北京语言大学信息科学学院的团队，由一名指导老师和七名学生组成，正致力于一项大学生创新创业项目。我们的目标是完成一个包含上千条多轮对话记录的数据库，以供自然语言处理及语用学等方向的研究使用。

理论基础

人类对话是重要的语言数据，在学术层面对中文多轮对话的研究相对较少。尽管语言大模型能够生成多轮对话，但仍面临真实对话复杂性和语境理解的挑战。人类的研究有助于深入理解对话中的语言现象、交互模式和情感表达，推动对话系统和自然语言处理的发展。此外，对指代消解、语用推断和上下文理解等细粒度问题的研究将改善多轮对话系统性能，适应不同对话场景。人类研究在学术层面仍具有独特价值，推动对话理解和生成领域的发展。

隐私保护

在聊天数据使用前，我们会采用包括内容替换、加密脱敏等多种数据脱敏技术增强个人信息在使用中安全性。您将受到保护的信息包括但不限于账号、密码、用户名；人名、昵称、地名、组织机构名等。经过自动处理后，我们还会进行二次审查以确保第三方无法重新识别此类信息中的隐私内容。

语料采集

我们致力于获取多样化的多轮对话数据，为学术研究项目提供支持。通过收集日常对话记录和组织在线聊天室，我们将建立一个代表性的对话语料库。我们严格遵守隐私保护准则，并对个人信息进行脱敏处理。这些对话数据将为对话结构、语言使用和交互模式的研究提供宝贵资源，并为自然语言处理和对话系统的发展提供训练和评估材料。我们期待通过这一阶段的工作，为学术研究和技术创新做出贡献，推动对话理解和生成领域的进步。

可视化分析

我们将构建交互式的可视化界面，以便研究人员能够以多个角度深入研究对话数据，包括对话流程、参与者之间的转换、重要主题的出现频率等等。这将帮助研究人员发现对话中的模式和规律，并提供有关对话动态的全面视角。同时，可视化分析还将为语用学研究提供重要的洞察力。通过观察对话中的语言使用、言语行为和语用关系，我们能够进一步探索日常语言的特点和演化。这对于深入理解人类语言交流以及人机对话的实现具有重要意义。
